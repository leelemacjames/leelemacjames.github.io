PaperCornell

A tool for turning scientific essays into Cornell-notes worksheets, drawn from a Google Drive folder and rendered as ready-to-compile LaTeX.

1 · Credentials

Both values live only in this browser tab. Nothing is sent anywhere except directly to Google's Drive API and Anthropic's message endpoint.

Where do I get an Anthropic API key?
  1. Visit console.anthropic.com, sign in, and add a payment method under Settings → Billing.
  2. Go to Settings → API Keys → Create Key, name it, and copy the sk-ant-... string immediately, since it is only shown once.
  3. Paste it into the field above; it is stored in this tab's sessionStorage and cleared when the tab closes.

2 · Source

Path to a folder in your Google Drive, relative to My Drive. Any PDF inside is processed. Sub-folders are ignored; if you want sub-folders processed, run the tool once per folder.

3 · Run

Each PDF in the folder is processed sequentially: text extracted in your browser via pdf.js, sent to Claude for question generation, then rendered into a LaTeX source file offered for download below. The compile step (pdflatex) happens off-tool — download the .tex, then compile locally or drop it into Overleaf.
