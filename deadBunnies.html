The exact mortality operator

This publication-facing version makes the exploratory UI mathematically explicit. The spectral plot, matrix heatmap, and projective monitor are derived from the same age-structured mortality operator rather than from a proxy recurrence.

For finite g, define the age-state vector

State vector
v_n = (a_0(n), a_1(n), …, a_{g-1}(n))^⊤

where a₀ is the number of newborn bunnies and a₁,…,ag−1 are rabbit age classes. The lifecycle rule becomes a single linear update

Mortality operator
v_{n+1} = M_g v_n

with the exact transition matrix

Exact matrix
M_g = [birth row from a₁,…,a_{g−1}; bunny maturation; rabbit aging subdiagonal]
  1. Birth row. Every rabbit class a₁,…,ag−1 contributes one newborn to the next step.
  2. Maturation. The bunny class a₀ becomes the first reproductive class a₁.
  3. Aging. Each rabbit class shifts forward by one age step along the subdiagonal.
  4. Death. There is no outgoing survival term from the oldest class ag−1.

Projective monitor

The live projective observable is computed from four consecutive total populations

Cross-ratio observable
ξ_n = ((x_{n-3}-x_{n-1})(x_{n-2}-x_n))/((x_{n-3}-x_n)(x_{n-2}-x_{n-1}))

and compared with a spectral estimate built from the dominant and subdominant eigenvalues of M_g. If λ₁ is dominant and r = λ₂ / λ₁, the monitor uses

Asymptotic estimate
ξ_∞ ≈ (1+r)^2 / (1+r+r^2)
Interpretive bridge

The spiral shows the visible tiling. The matrix shows the memory mechanism. The spectrum shows the decay structure. The projective monitor shows convergence in a form the user can watch generation by generation.
