Beatty Sequences

Partitioning ℕ with Irrationals · k-Nacci Ratios · √k Extensions
Pick any irrational α > 1. Let β = α/(α−1). Then ⌊nα⌋ and ⌊nβ⌋ partition the positive integers. No overlaps, no gaps. Beatty's theorem, 1926.

Beatty's Theorem

If α is a positive irrational number greater than 1, and β is defined by 1/α + 1/β = 1, then the sequences ⌊nα⌋ and ⌊nβ⌋ for n = 1, 2, 3, … partition the positive integers.

1/α + 1/β = 1
β = α/(α − 1)
Beatty pair: {⌊nα⌋} ∪ {⌊nβ⌋} = ℕ, with the union disjoint

This is remarkable. Every positive integer appears in exactly one of the two sequences. No integer is missed, no integer appears twice. The only condition: α must be irrational (and > 1). If α were rational, some integers would be hit by both sequences or missed by both.

The golden ratio case (your slide)

When α = φ = (1+√5)/2 ≈ 1.618…, then β = φ/(φ−1) = φ² ≈ 2.618… (using the identity φ² = φ + 1). The two Beatty sequences are:

ℤφ = ⌊nφ⌋ = 1, 3, 4, 6, 8, 9, 11, 12, 14, 16, 17, 19, 21, …

ℤ*φ = ⌊nφ²⌋ = 2, 5, 7, 10, 13, 15, 18, 20, 23, 26, 28, …

Together they cover every positive integer exactly once.

Verification

The Irrationals

Different families of irrationals give different Beatty pairs. The k-nacci constants generalise the golden ratio; the √k family connects to quadratic irrationals.

k-Nacci metallic constants

The k-nacci constant is the positive root of xk = xk−1 + xk−2 + … + 1. For k=2 this is the golden ratio φ; for k=3 the tribonacci constant; for k=4 the tetranacci constant. As k → ∞, the constant approaches 2.

√k irrationals

For any non-square integer k, √k is irrational. The Beatty complement is β = √k/(√k − 1). These give a different flavour of partition: the gap structure depends on the continued fraction expansion of √k.

Why must α be irrational? If α = p/q were rational, then ⌊nα⌋ would eventually produce the same integer as ⌊mβ⌋ for some n, m. The proof uses the fact that nα is never an integer when α is irrational, which prevents boundary collisions.

Proof of Beatty's Theorem

We show that for each positive integer k, exactly one of the two sequences hits k.

01Let 1/α + 1/β = 1, with α, β irrational and > 1.
02Define A(k) = #{n : ⌊nα⌋ ≤ k} and B(k) = #{n : ⌊nβ⌋ ≤ k}.count how many terms ≤ k
03⌊nα⌋ ≤ k ⟺ nα < k+1 ⟺ n < (k+1)/α.since nα is never integer
04So A(k) = ⌊k/α + 1/α − ε⌋ for tiny ε, which equals ⌊(k+1)/α⌋ − 1 + 1 = ⌊(k+1)/α⌋.
05Wait, more carefully: A(k) = #{n ≥ 1 : n ≤ (k+1)/α − ε} = ⌊(k+1)/α⌋ if (k+1)/α ∉ ℤ.
06Similarly B(k) = ⌊(k+1)/β⌋.
07A(k) + B(k) = ⌊(k+1)/α⌋ + ⌊(k+1)/β⌋.
08Since 1/α + 1/β = 1: (k+1)/α + (k+1)/β = k+1.
09Since neither (k+1)/α nor (k+1)/β is an integer (α, β irrational):
10⌊x⌋ + ⌊y⌋ = x + y − {x} − {y} where {·} is the fractional part.
11Since {x} + {y} ∈ (0,2) and {x}+{y} = x+y − ⌊x⌋−⌊y⌋, and x+y = k+1 ∈ ℤ:
12{x} + {y} must be exactly 1 (the only integer value in (0,2) that makes ⌊x⌋+⌊y⌋ integer).
∴A(k) + B(k) = (k+1) − 1 = k. Exactly k terms ≤ k. QED.

The key move: Since α and β are both irrational, (k+1)/α and (k+1)/β are both non-integer. Their fractional parts sum to exactly 1 (because their sum is the integer k+1). So ⌊(k+1)/α⌋ + ⌊(k+1)/β⌋ = k+1 − 1 = k. This means exactly k terms from the two sequences combined are ≤ k, which forces a perfect partition.

Python

import math def beatty_pair(alpha, N): """Generate Beatty sequences for alpha and its complement.""" beta = alpha / (alpha - 1) A = [math.floor(n * alpha) for n in range(1, N+1)] B = [math.floor(n * beta) for n in range(1, N+1)] return A, B, beta # k-nacci constant: positive root of x^k = x^(k-1) + ... + 1 def knacci_constant(k, tol=1e-12): """Newton's method for the k-nacci constant.""" x = 1.9 # initial guess near 2 for _ in range(100): # f(x) = x^k - x^(k-1) - ... - 1 = x^k - (x^(k-1)-1)/(x-1) num = x**k - (x**(k) - 1) / (x - 1) # = x^k - sum(x^i, i=0..k-1) # derivative dnum = k*x**(k-1) - sum(i*x**(i-1) for i in range(1,k)) x -= num / dnum return x # Golden ratio example phi = (1 + math.sqrt(5)) / 2 A, B, beta = beatty_pair(phi, 20) print(f"α = φ = {phi:.6f}, β = φ² = {beta:.6f}") print(f"A = {A}") print(f"B = {B}") # Verify partition combined = sorted(A + B) assert combined == list(range(1, combined[-1]+1)), "Not a partition!" print("✓ Perfect partition verified")

Postamble

Why this matters

Beatty's theorem sits at a crossroads of number theory, combinatorics, and the theory of continued fractions. The partition it describes is not just a curiosity: it appears in Wythoff's game (a combinatorial game where the losing positions are exactly the Beatty sequence of φ), in the theory of quasicrystals (where the gap sequence of ⌊nφ⌋ generates the Fibonacci word), and in the distribution of electrons in the quantum Hall effect.

The Fraenkel conjecture

Beatty's theorem partitions ℕ into two sequences. Fraenkel generalised the question: can we partition ℕ into k ≥ 3 Beatty-like sequences? The answer is yes, but the conditions become much more restrictive. For three sequences, the only known solution uses α, β, γ with specific algebraic relationships. The full characterisation for k ≥ 4 remains open.

k-Nacci convergence to 2

As k increases, the k-nacci constant approaches 2 from below. This means the Beatty complement β = α/(α−1) approaches 2/(2−1) = 2 from above. In the limit, both sequences try to be "every other integer," but the irrationality prevents exact dovetailing, creating increasingly subtle gap patterns. The tribonacci constant ≈ 1.839 already gives a noticeably different partition structure from φ ≈ 1.618.

Connection to the Stern sequence

The rationals S(n)/S(n+1) from the Stern-Brocot sequence enumerate ℚ⁺. Beatty's theorem partitions ℕ using irrationals. Together they sketch a picture: the rationals tile the number line densely, while each irrational carves ℕ into two complementary combs. The continued fraction of α determines the gap pattern of the Beatty sequence, and the mediants of the Stern-Brocot tree approximate α from both sides.

The Fibonacci word

The gap sequence of the Beatty sequence of φ (the differences ⌊(n+1)φ⌋ − ⌊nφ⌋) is either 1 or 2, and the pattern of 1s and 2s is the Fibonacci word: a non-periodic, self-similar sequence that is the 1D analogue of a Penrose tiling. This connects Beatty sequences to aperiodic order, quasicrystals, and the cut-and-project method in mathematical physics.
