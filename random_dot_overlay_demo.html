Random dot overlay

Two independent random-dot fields are drawn on the same canvas. The second field can be rotated or rescaled. This lets you explore alignment effects, drift, and moiré-like structure from random point overlays.

Animation speed: 0.40
In rotate mode, field B is spun about the canvas centre. In scale mode, field B is dilated about the same centre, so 0.96 means a 4% contraction relative to field A. In translate mode, field B is shifted rigidly in the plane. Set field B to be a copy of field A to get the strongest visual effects.
Field A Fixed random dots
Field B Rotation 0.0°
The two fields are drawn in different tones so the transformed overlay is visible. For a cleaner visual analogy to printed transparencies, you can push the dot size down and the count up.
