2D Ising Model — spin lattice with energy, specific heat, external field and hysteresis.

THE MODEL
ℋ = −J Σ⟨i,j⟩ sisj − Hext Σi si    where si ∈ {+1, −1}
Exchange term (−J)
Nearest neighbours want to align. Both match → energy −J; opposed → +J. J>0 makes ferromagnetism favourable.
Zeeman term (−H)
An external field biases every spin toward ↑ — exactly the paramagnetism you know: individual moments aligning with an applied field.
Metropolis algorithm
Pick a random spin, propose a flip. Accept if ΔE≤0, or with probability e−ΔE/T if ΔE>0. This samples the Boltzmann distribution at temperature T.

WHAT THE CHARTS SHOW
M
Magnetisation |M| — order parameter. Near 1 = aligned (ferromagnet). Near 0 = disordered (paramagnet). Collapses sharply above Tc.
E
Energy/spin — from −2J (fully aligned) toward 0 (fully random). Rises as T disorders the lattice.
C
Specific heat C = ⟨ΔE²⟩/T² — peaks sharply at Tc. Drag T slowly through 2.27 while running to trace the spike.
H
Hysteresis — press Sweep H below Tc. The M–H loop shows remanence and coercivity. Above Tc the loop collapses to a line.
EXPERIMENTS TO TRY
1. Quench from disorder
Set T = 3, initialise random, press Run. Drag T down to 1. Domains nucleate and merge — a ferromagnet cooling through its Curie temperature.
2. Find Tc via specific heat
Switch to C tab. Drag T slowly 1→4 while running. The spike locates Tc exactly — the same method as laboratory calorimetry.
3. Hysteresis loop
Set T = 1.5, press Sweep H. Read coercive field (M crosses zero) and remanence (M at H = 0). Repeat at T = 3 — loop vanishes.
