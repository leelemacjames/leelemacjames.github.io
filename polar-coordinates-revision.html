01 Coordinate Conversions
Q1

State the relationships between polar coordinates $(r, \theta)$ and Cartesian coordinates $(x, y)$. Why must care be taken with $\theta = \arctan(y/x)$?

Core
$$x = r\cos\theta, \qquad y = r\sin\theta$$
$$r^2 = x^2 + y^2, \qquad \theta = \arctan\!\left(\frac{y}{x}\right)$$

The issue with $\arctan(y/x)$ is that it only returns values in $\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)$, so it can't distinguish between a point in the first quadrant and one diametrically opposite in the third — the ratio $y/x$ is the same for both. You need to check which quadrant $(x,y)$ sits in and adjust $\theta$ accordingly (typically by adding $\pi$). In Python, math.atan2(y, x) handles this for you.

Q2

Convert the polar equation $r = \cos\theta + \sin\theta$ into Cartesian form. Hence show it represents a circle, and find its centre and radius.

Core
01

Start with $r = \cos\theta + \sin\theta$. Multiply both sides by $r$:

$$r^2 = r\cos\theta + r\sin\theta$$
02

Substitute $r^2 = x^2+y^2$, $r\cos\theta = x$, $r\sin\theta = y$:

$$x^2 + y^2 = x + y$$
03

Complete the square in both variables:

$$\left(x - \tfrac{1}{2}\right)^2 + \left(y - \tfrac{1}{2}\right)^2 = \tfrac{1}{2}$$

This is a circle with centre $\left(\frac{1}{2}, \frac{1}{2}\right)$ and radius $\frac{\sqrt{2}}{2}$.

The "multiply both sides by $r$" trick is the standard opening move for converting polar equations to Cartesian form. Memorise it.
02 Recognising Standard Curves
Q3

What curve does $r = a$ represent? What about $\theta = \alpha$ and $r = a\theta$?

Core

$r = a$ is a circle of radius $a$ centred at the pole.

$\theta = \alpha$ is a half-line (ray) from the pole at angle $\alpha$ to the initial line.

$r = a\theta$ is an Archimedean spiral — the radius grows linearly with angle, so successive windings are equally spaced.

Q4

Describe the curves given by $r = \sin(n\theta)$ and $r = \cos(n\theta)$ for positive integer $n$. How many petals does each rose have?

Core

These are rose curves. The number of petals depends on whether $n$ is odd or even:

If $n$ is odd, the curve has exactly $n$ petals.

If $n$ is even, the curve has $2n$ petals.

For example, $r = \cos 3\theta$ gives a 3-petal rose, while $r = \sin 4\theta$ gives an 8-petal rose. The $\sin$ vs $\cos$ choice just rotates the figure.

The special cases $r = \sin\theta$ and $r = \cos\theta$ (i.e. $n=1$) each give a single circle — technically a "1-petal rose."

Q5

What are lemniscates? Give the general form and describe the shape.

Core

Lemniscates have the form $r^2 = a^2\sin(n\theta)$ or $r^2 = a^2\cos(n\theta)$. They look like figure-of-eight curves (for $n=2$), similar to the symbol $\infty$.

Because $r^2$ must be non-negative, these curves only exist where $\sin(n\theta) \geq 0$ (or $\cos(n\theta) \geq 0$), so they have natural gaps — the curve doesn't trace out for all $\theta$.

If you see $r^2 = \ldots$ in a polar equation, think lemniscate. The $r^2$ form also means you'll need to take care with signs when computing $r$.
Q6

For curves of the form $r = p + q\cos\theta$, describe how the shape changes depending on the relative sizes of $p$ and $q$.

Apply

These are limaçons. Three cases:

$p \geq 2q$: A convex, roughly oval shape (no dimple, no loop).

$q < p < 2q$: A "dimpled" limaçon — there's an inward dent but no self-intersection.

$p = q$: A cardioid — the curve passes through the pole and has a cusp there.

If $p < q$, the curve has an inner loop that passes through the pole.

The exam problem $r = a(4 + 3\cos\theta)$ has $p=4a$, $q=3a$, so $q < p < 2q$. That's a dimpled limaçon — the dimple appears at $\theta = \pi$.
Q7

Sketch and describe the curve $r = 2\sec\!\left(\theta - \frac{\pi}{3}\right)$. What does it look like in Cartesian coordinates?

Apply
01

Rewrite: $r = \frac{2}{\cos(\theta - \pi/3)}$, so $r\cos(\theta - \pi/3) = 2$.

02

Expand using the compound angle formula:

$$r\left(\cos\theta\cos\frac{\pi}{3} + \sin\theta\sin\frac{\pi}{3}\right) = 2$$
$$\frac{1}{2}x + \frac{\sqrt{3}}{2}y = 2$$

This is a straight line. Any polar equation of the form $r = a\sec(\theta - \alpha)$ represents a line perpendicular to the direction $\theta = \alpha$, at distance $a$ from the pole.

03 Area of a Polar Region
Q8

State the formula for the area of a sector bounded by a polar curve $r = f(\theta)$ between $\theta = \alpha$ and $\theta = \beta$. Where does the $\frac{1}{2}$ come from?

Core
$$A = \frac{1}{2}\int_\alpha^\beta r^2\,\mathrm{d}\theta$$

The $\frac{1}{2}$ comes from the area of a thin circular sector: a sector of radius $r$ and infinitesimal angle $\mathrm{d}\theta$ has area $\frac{1}{2}r^2\,\mathrm{d}\theta$, just as a full circle of radius $r$ has area $\frac{1}{2}r^2 \cdot 2\pi = \pi r^2$. We're summing up infinitely many such thin wedges.

Q9

Find the area enclosed by the curve $r = 2\cos\theta + 2\sin\theta$ between $\theta = \frac{\pi}{6}$ and $\theta = \frac{\pi}{2}$.

Apply
01

Set up the integral. We need $\frac{1}{2}\int_{\pi/6}^{\pi/2}(2\cos\theta+2\sin\theta)^2\,\mathrm{d}\theta$.

02

Expand the square: $(2\cos\theta+2\sin\theta)^2 = 4(\cos\theta+\sin\theta)^2 = 4(1+2\sin\theta\cos\theta) = 4(1+\sin 2\theta)$.

$$A = \frac{1}{2}\int_{\pi/6}^{\pi/2}4(1+\sin 2\theta)\,\mathrm{d}\theta = 2\int_{\pi/6}^{\pi/2}(1+\sin 2\theta)\,\mathrm{d}\theta$$
03

Integrate:

$$= 2\left[\theta - \frac{\cos 2\theta}{2}\right]_{\pi/6}^{\pi/2} = 2\left[\left(\frac{\pi}{2} - \frac{\cos\pi}{2}\right) - \left(\frac{\pi}{6} - \frac{\cos(\pi/3)}{2}\right)\right]$$
$$= 2\left[\left(\frac{\pi}{2}+\frac{1}{2}\right)-\left(\frac{\pi}{6}-\frac{1}{4}\right)\right] = 2\left(\frac{\pi}{3}+\frac{3}{4}\right) = \frac{2\pi}{3}+\frac{3}{2} \approx 3.59$$
Q10

The curves $r = a\sin 4\theta$ and $r = a\sin 2\theta$ are given for $0 \leq \theta \leq \frac{\pi}{2}$. The region $R$ is contained within both curves. Find the area of $R$ in terms of $a$.

Exam
01

Find intersection points. Set $a\sin 4\theta = a\sin 2\theta$, so $\sin 4\theta = \sin 2\theta$.

This gives $4\theta = 2\theta + 2k\pi$ or $4\theta = \pi - 2\theta + 2k\pi$, yielding $\theta = k\pi$ or $\theta = \frac{\pi}{6}(1+2k)$. In our range: $\theta = 0$, $\theta = \frac{\pi}{6}$, $\theta = \frac{\pi}{2}$.

02

Determine which curve is "inner" on each sub-interval. Checking a test point shows $a\sin 2\theta$ is the inner (smaller $r$) curve for $0 \leq \theta \leq \frac{\pi}{6}$, and $a\sin 4\theta$ is inner for $\frac{\pi}{6} \leq \theta \leq \frac{\pi}{4}$ (beyond $\frac{\pi}{4}$, $\sin 4\theta < 0$ so that loop doesn't contribute).

03

Set up the area as the sum of two half-integrals of $r^2$ over the inner curve on each piece:

$$A = \frac{1}{2}\int_0^{\pi/6}(a\sin 2\theta)^2\,\mathrm{d}\theta + \frac{1}{2}\int_{\pi/6}^{\pi/4}(a\sin 4\theta)^2\,\mathrm{d}\theta$$
04

Use $\sin^2 u = \frac{1}{2}(1-\cos 2u)$ on each piece, integrate, and evaluate:

$$A = \frac{a^2}{4}\left(\frac{\pi}{6} - \frac{\sqrt{3}}{8}\right) + \frac{a^2}{4}\left(\frac{\pi}{4} - \frac{\pi}{6} - \frac{\sqrt{3}}{16}\right)$$

Combining:

$$A = \frac{a^2}{4}\left(\frac{\pi}{4} - \frac{3\sqrt{3}}{16}\right)$$
The trick with "area inside both curves" is that you're integrating the smaller of the two $r$-values on each sub-interval. Sketch first to see which curve is inner where.
04 Tangent Lines — Parallel & Perpendicular to the Initial Line
Q11

State the conditions for the tangent to a polar curve to be (i) parallel to the initial line, and (ii) perpendicular to it. Why do we use $x = r\cos\theta$ and $y = r\sin\theta$ rather than $\frac{\mathrm{d}r}{\mathrm{d}\theta}$?

Core

(i) Tangent parallel to initial line (horizontal tangent): set $\frac{\mathrm{d}y}{\mathrm{d}\theta} = 0$.

(ii) Tangent perpendicular to initial line (vertical tangent): set $\frac{\mathrm{d}x}{\mathrm{d}\theta} = 0$.

We can't just use $\frac{\mathrm{d}r}{\mathrm{d}\theta} = 0$ because that gives turning points in the radial direction (where $r$ is locally max or min), which is a different geometric condition. The tangent direction in the Cartesian plane is governed by $\frac{\mathrm{d}y}{\mathrm{d}x}$, and the chain rule gives $\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\mathrm{d}y/\mathrm{d}\theta}{\mathrm{d}x/\mathrm{d}\theta}$.

Q12

Find the point on the curve $r = a(1 + \tan\theta)$, $0 \leq \theta < \frac{\pi}{2}$, where the tangent is perpendicular to the initial line.

Exam
01

Perpendicular to the initial line means a vertical tangent, so we need $\frac{\mathrm{d}x}{\mathrm{d}\theta} = 0$.

02

Write $x = r\cos\theta = a(1+\tan\theta)\cos\theta = a(\cos\theta + \sin\theta)$.

03

Differentiate: $\frac{\mathrm{d}x}{\mathrm{d}\theta} = a(-\sin\theta + \cos\theta)$.

04

Set equal to zero: $\cos\theta = \sin\theta$, so $\tan\theta = 1$, giving $\theta = \frac{\pi}{4}$.

05

Substitute back: $r = a(1 + \tan\frac{\pi}{4}) = a(1+1) = 2a$.

$$\text{The point is } \left(2a,\; \frac{\pi}{4}\right)$$
Notice how $a(1+\tan\theta)\cos\theta$ simplifies beautifully to $a(\cos\theta+\sin\theta)$ — always simplify before differentiating.
Q13

The curve $C$ has polar equation $r = 1 + 3\cos\theta$, $0 \leq \theta \leq \frac{\pi}{2}$. The tangent at a point $A$ on $C$ is parallel to the initial line. Point $O$ is the pole. Find the exact length $OA$.

Exam
01

Parallel to the initial line means $\frac{\mathrm{d}y}{\mathrm{d}\theta} = 0$. Write $y = r\sin\theta = (1+3\cos\theta)\sin\theta = \sin\theta + 3\cos\theta\sin\theta$.

02

Differentiate: $\frac{\mathrm{d}y}{\mathrm{d}\theta} = \cos\theta + 3(\cos^2\theta - \sin^2\theta) = \cos\theta + 3\cos 2\theta$.

Rewrite using $\cos 2\theta = 2\cos^2\theta - 1$:

$$\cos\theta + 6\cos^2\theta - 3 = 0$$
03

This is a quadratic in $\cos\theta$. Let $c = \cos\theta$: $6c^2 + c - 3 = 0$.

$$c = \frac{-1 \pm \sqrt{1 + 72}}{12} = \frac{-1 \pm \sqrt{73}}{12}$$
04

Since $0 \leq \theta \leq \frac{\pi}{2}$, we need $c \geq 0$, so $\cos\theta = \frac{-1+\sqrt{73}}{12}$.

05

The length $OA$ is just $r$ at this $\theta$:

$$r = 1 + 3\cdot\frac{-1+\sqrt{73}}{12} = 1 + \frac{-3+3\sqrt{73}}{12} = \frac{12-3+3\sqrt{73}}{12} = \frac{9+3\sqrt{73}}{12} = \frac{3+\sqrt{73}}{4}$$
The quadratic-in-$\cos\theta$ pattern appears constantly in tangent problems. Get comfortable with it.
05 Mixed Exam Problems
Q14

Show that the locus $|z - 1 - \mathrm{i}| = \sqrt{2}$ on the Argand diagram can be represented by the polar curve $r = 2\cos\theta + 2\sin\theta$.

Exam
01

$|z-1-\mathrm{i}|=\sqrt{2}$ is a circle centred at $(1,1)$ with radius $\sqrt{2}$. In Cartesian form: $(x-1)^2+(y-1)^2=2$.

02

Expand: $x^2 - 2x + 1 + y^2 - 2y + 1 = 2$, so $x^2 + y^2 = 2x + 2y$.

03

Substitute $x^2+y^2 = r^2$, $x = r\cos\theta$, $y = r\sin\theta$:

$$r^2 = 2r\cos\theta + 2r\sin\theta$$

Divide by $r$ (valid for $r \neq 0$):

$$r = 2\cos\theta + 2\sin\theta \qquad \square$$
Q15

The set $A = \left\{z : \frac{\pi}{6} \leq \arg z \leq \frac{\pi}{2}\right\} \cap \left\{z : |z-1-\mathrm{i}| \leq \sqrt{2}\right\}$. Find, correct to 3 s.f., the area of the region defined by $A$.

Exam

From Q14, the circle in polar form is $r = 2\cos\theta + 2\sin\theta$. The set $A$ is the portion of the disk where $\frac{\pi}{6} \leq \theta \leq \frac{\pi}{2}$.

01

Use the polar area formula with the result from Q9:

$$A = \frac{1}{2}\int_{\pi/6}^{\pi/2}(2\cos\theta+2\sin\theta)^2\,\mathrm{d}\theta = 2\left(\frac{\pi}{3}+\frac{3}{4}\right) \approx 3.59 \text{ (3 s.f.)}$$
This question is a classic combo: complex numbers → Cartesian circle → polar form → polar area integral. The examiners love linking these topics.
Q16

The curve $C$ has polar equation $r = 1 + \cos 3\theta$, $0 \leq \theta \leq \frac{\pi}{3}$. At points $A$ and $B$, $r = \frac{2+\sqrt{2}}{2}$. Point $A$ lies on $C$ and point $B$ lies on the initial line. Find, correct to 3 s.f., the finite area bounded by the curve, the line segment $AB$, and the initial line.

Exam
01

Find $\theta_A$: Solve $1+\cos 3\theta = \frac{2+\sqrt{2}}{2}$, giving $\cos 3\theta = \frac{\sqrt{2}}{2}$, so $3\theta = \frac{\pi}{4}$ and $\theta_A = \frac{\pi}{12}$.

02

Sector area (from $\theta=0$ to $\theta=\frac{\pi}{12}$, bounded by curve and line $OA$):

$$A_{\text{sector}} = \frac{1}{2}\int_0^{\pi/12}(1+\cos 3\theta)^2\,\mathrm{d}\theta$$

Expand: $(1+\cos 3\theta)^2 = 1 + 2\cos 3\theta + \cos^2 3\theta = 1 + 2\cos 3\theta + \frac{1}{2}(1+\cos 6\theta)$.

$$= \frac{1}{2}\left[\frac{3}{2}\theta + \frac{2}{3}\sin 3\theta + \frac{1}{12}\sin 6\theta\right]_0^{\pi/12}$$
$$= \frac{1}{2}\left(\frac{\pi}{8} + \frac{\sqrt{2}}{3} + \frac{1}{12}\right) \approx 0.4737$$
03

Triangle $OAB$ area: Both $OA$ and $OB$ have length $\frac{2+\sqrt{2}}{2}$, with angle $\frac{\pi}{12}$ between them.

$$A_{\triangle} = \frac{1}{2}\left(\frac{2+\sqrt{2}}{2}\right)^2\sin\frac{\pi}{12} \approx 0.3771$$
04

Shaded area = sector area − triangle area $\approx 0.4737 - 0.3771 = 0.0966$ (3 s.f.).

"Area between curve and chord" problems always decompose as: polar sector area minus triangle area. The triangle uses the $\frac{1}{2}ab\sin C$ formula with the two radii and the angle between them.
Q17

Describe the curve $\theta = -\frac{\pi}{4}$. Sketch it, and explain its relationship to the Cartesian equation $y = -x$.

Apply

$\theta = -\frac{\pi}{4}$ is a half-line from the pole in the direction $-45°$ (i.e. into the fourth quadrant). It is not the full line $y = -x$ — only the ray from the origin going towards the bottom-right.

The full line $y = -x$ would require both $\theta = -\frac{\pi}{4}$ and $\theta = \frac{3\pi}{4}$ (or equivalently, allowing negative $r$). This is a common subtlety: a single $\theta = \alpha$ equation only gives a half-line, not a full line.

Q18

The curve $r = a\cos 3\theta$ is a 3-petalled rose. For what range of $\theta$ does the first petal (the one along the initial line) trace out? What is the area of this single petal?

Apply

The first petal lies along $\theta = 0$ and is traced for $-\frac{\pi}{6} \leq \theta \leq \frac{\pi}{6}$ (where $\cos 3\theta \geq 0$, so $r \geq 0$).

$$A = \frac{1}{2}\int_{-\pi/6}^{\pi/6}a^2\cos^2 3\theta\,\mathrm{d}\theta = \frac{a^2}{2}\int_{-\pi/6}^{\pi/6}\frac{1+\cos 6\theta}{2}\,\mathrm{d}\theta$$
$$= \frac{a^2}{4}\left[\theta + \frac{\sin 6\theta}{6}\right]_{-\pi/6}^{\pi/6} = \frac{a^2}{4}\cdot\frac{\pi}{3} = \frac{\pi a^2}{12}$$

The total area of all three petals is $\frac{\pi a^2}{4}$.

Q19

For the lemniscate $r^2 = a^2\sin 2\theta$, explain why the curve only exists for $0 \leq \theta \leq \frac{\pi}{2}$ and $\pi \leq \theta \leq \frac{3\pi}{2}$. What does the full curve look like?

Apply

Since $r^2 = a^2\sin 2\theta$ and $r^2 \geq 0$, we need $\sin 2\theta \geq 0$. This holds when $2\theta \in [0, \pi] \cup [2\pi, 3\pi]$, i.e. $\theta \in [0, \pi/2] \cup [\pi, 3\pi/2]$.

The curve is a figure-of-eight with two lobes: one in the first quadrant, one in the third. It passes through the origin at $\theta = 0$ and $\theta = \pi/2$.

For lemniscate area problems, the area of one lobe is $\frac{1}{2}\int_0^{\pi/2}a^2\sin 2\theta\,\mathrm{d}\theta = \frac{a^2}{2}$, so both lobes together give $a^2$.
Q20

A student claims that $r = a(4+3\sin\theta)$ is "the same shape as $r = a(4+3\cos\theta)$ but rotated by $90°$." Justify or refute this claim, and describe the key features of the curve $r = a(4+3\sin\theta)$.

Exam

The student is correct. Replacing $\cos\theta$ with $\sin\theta$ is equivalent to replacing $\theta$ with $\theta - \frac{\pi}{2}$ (since $\sin\theta = \cos(\theta - \pi/2)$). This rotates the curve by $\frac{\pi}{2}$ anticlockwise.

Both curves are dimpled limaçons (since $3a < 4a < 2\times 3a$, i.e. $q < p < 2q$).

Key features of $r = a(4+3\sin\theta)$:

Maximum $r = 7a$ at $\theta = \frac{\pi}{2}$ (top of curve). Minimum $r = a$ at $\theta = \frac{3\pi}{2}$ (bottom). The dimple appears at the bottom. The axis of symmetry is now the line $\theta = \frac{\pi}{2}$ (the $y$-axis) rather than the initial line.

$\cos\theta \to \sin\theta$ is always a $90°$ rotation. Similarly, $\cos n\theta \to \sin n\theta$ rotates the curve by $\frac{\pi}{2n}$.
