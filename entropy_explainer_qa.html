§ 01

What is Entropy?

Entropy is a thermodynamic state function measuring how energy is dispersed across the available microstates of a system. It is not simply "disorder" — that is a useful intuition but imprecise. More accurately: entropy counts, logarithmically, the number of ways a system can be arranged at the microscopic level while presenting the same macroscopic appearance.

S = kB ln Ω Boltzmann's entropy formula · Ω = number of accessible microstates · kB = 1.38 × 10⁻²³ J K⁻¹

The key insight: a macrostate is what we observe (temperature, pressure, volume). A microstate is one specific configuration of all particles consistent with that macrostate. High-entropy macrostates are overwhelmingly probable simply because many more microstates realise them.

The Clausius Definition

In classical thermodynamics, for a reversible process, entropy change is defined by heat flow:

ΔS = Q / T Valid for reversible (quasi-static) processes · Q in joules · T in kelvin

For irreversible processes, ΔSuniverse > 0. For reversible ones, ΔSuniverse = 0. Entropy can locally decrease (a refrigerator cools its interior) but must increase globally.

The Second Law — Three Formulations

Clausius
Heat does not spontaneously flow from a cold body to a hot body.
Kelvin–Planck
No heat engine operating in a cycle can convert all its input heat into work.
Entropy
The total entropy of an isolated system never decreases. For irreversible processes it strictly increases; for reversible ones it remains constant.

Non-isolated Systems

A non-isolated system can have its entropy decrease — but only if the surroundings gain at least as much entropy. Living organisms, refrigerators, and crystals forming from solution all locally decrease entropy, but they export it to their environment. The second law governs the total entropy of system plus surroundings.

Key distinction
Isolated system: ΔS ≥ 0 always.
Non-isolated system: ΔS can be negative, positive, or zero — but ΔStotal = ΔSsystem + ΔSsurroundings ≥ 0.
§ 02

Heat Engines & Efficiency

A heat engine operates between a hot reservoir at TH and a cold reservoir at TC, absorbing heat QH, doing work W, and rejecting heat QC to the cold reservoir. Energy conservation (first law) gives:

W = QH − QC Work output = heat absorbed − heat rejected

The thermal efficiency is the fraction of absorbed heat converted to useful work:

η = W / QH = 1 − QC/QH Always < 1 for any real engine

The Carnot Limit

The second law places an absolute ceiling on efficiency. The ideal (Carnot) engine — operating reversibly — achieves the maximum possible efficiency for given reservoir temperatures:

ηCarnot = 1 − TC / TH Temperatures must be in kelvin · This is the theoretical maximum

Any real engine has η < ηCarnot because real processes are irreversible — friction, finite temperature differences, turbulence all generate entropy and waste energy as heat.

Identifying Processes

Adiabatic identification trick
On a pV diagram, an adiabatic curve is steeper than an isotherm passing through the same point. This is because pVγ = const (adiabatic) vs pV = const (isothermal), and γ > 1. If two curves connect the same two pressure levels but one covers a greater volume range, the steeper one is adiabatic.

Entropy Changes in the Cycle

Over a complete cycle, the gas returns to its initial state, so ΔSgas = 0 for the full cycle. Individual processes:

  • Adiabatic: Q = 0 → ΔS = 0 (if reversible)
  • Isothermal expansion (A→B in the paper's cycle): Heat flows in → ΔS = Qin/T > 0, entropy increases
  • Isovolumetric cooling (B→C): Heat flows out → ΔS = Qout/T < 0, entropy decreases

Second Law Consistency Check

Given TH = 602 K and TC = 92 K, the Carnot limit is:

ηCarnot = 1 − 92/602 ≈ 0.847 The actual engine runs at η ≈ 0.6, which is below this ceiling — fully consistent with the second law
§ 03

Microstates, Macrostates & Irreversibility

Consider N distinguishable particles that can each occupy one of two halves of a box. Each particle's half-assignment is one binary "choice." A microstate specifies which half each individual particle is in. A macrostate specifies only the numbers in each half — not which particles.

Counting Microstates

If all N particles are in the left half (configuration A), there is exactly one microstate: the only assignment is {L, L, L, …, L}. If N/2 particles are in each half (configuration B), the number of microstates is:

ΩB = C(N, N/2) = N! / ((N/2)!)² For N = 20: ΩB = 184,756 vs ΩA = 1

This is why gases expand — not because they are "driven" to, but because the equal-split macrostate is astronomically more probable. The system random-walks through microstates, and almost all of them look like the equal-split macrostate.

Why Expansion is Irreversible

The irreversibility argument
Starting in configuration A (Ω = 1, S = kB ln 1 = 0), the gas expands into configuration B (Ω ≫ 1, S = kB ln Ω ≫ 0). Entropy increases. For the reverse to occur spontaneously, every particle would need to return to one half simultaneously — this has probability 1/ΩB, which is vanishingly small for macroscopic N. The second law is thus a statistical statement about overwhelming probability, not a fundamental prohibition.

Energy Microstates (Q5 of the paper)

Now suppose 6 particles share total energy 6E in integer units. Each assignment of energies to particles that sums to 6E is a microstate. Configuration A (one particle has all 6E) has exactly 6 microstates — any of the six particles can be the one holding all the energy. Configuration B (energies distributed as 3E, 2E, E, 0, 0, 0) has 120 microstates — counted by the number of ways to assign those values to 6 distinguishable particles.

ΔS = SB − SA = kB(ln 120 − ln 6) = kB ln 20 ≈ 3.0 kB System will evolve from A toward B: entropy increases, more microstates accessible

Boltzmann's Insight

S = kB ln Ω is the bridge between the statistical (counting microstates) and thermodynamic (heat flow, temperature) descriptions of entropy. The logarithm is essential: it makes entropy additive for independent systems. If system 1 has Ω₁ microstates and system 2 has Ω₂, the combined system has Ω₁ × Ω₂ microstates, and Stotal = kB ln(Ω₁Ω₂) = S₁ + S₂.
