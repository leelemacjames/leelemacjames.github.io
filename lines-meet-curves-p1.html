Three Kinds of Number

Every number lives somewhere on a line, but not all numbers are built the same way.

Integers..., −2, −1, 0, 1, 2, 3, ...
Rationals3/4, −7/5, 22/7, ...
Irrationals√2, φ, π, ...

Integers are the counting numbers (and their negatives). Rationals are every fraction of two integers. Irrationals are everything else: decimals that never terminate, never repeat. How can one tell which type a geometric construction produces?

The Quadratic and Its Discriminant

When a line meets a degree-two curve (a conic), the intersection points satisfy a quadratic equation:

ax² + bx + c = 0

The solutions come from:

x = (−b ± √(b² − 4ac)) / 2a

The expression under the square root is the discriminant:

Δ = b² − 4ac
When is Δ a perfect square? What kind of number does the formula produce when it is, and when it is not?

If Δ is a perfect square (0, 1, 4, 9, ...), then √Δ is an integer, and the formula yields rational roots. If Δ is positive but not a perfect square, √Δ is irrational, and so are the roots. The discriminant is the gatekeeper between the rational and irrational worlds.

The Unit Circle: Rational Intersections

The unit circle is x² + y² = 1. The point (−1, 0) lies on it. A line of slope m through that point is:

y = m(x + 1)
One intersection is always (−1, 0). Where is the other? What kind of number is it?

Substitute into x² + y² = 1:

x² + m²(x+1)² = 1  ⟹  (1+m²)x² + 2m²x + (m²−1) = 0

This factors as (x + 1) times a linear remainder, because x = −1 is guaranteed to be a root. The other root is:

x = (1 − m²)/(1 + m²)   y = 2m/(1 + m²)

If m = p/q is rational, then x and y are rational. Clear denominators: every Pythagorean triple emerges.

The Farey Sequence and Rational Slopes

The Farey sequence Fn is the ordered list of all fractions p/q in [0, 1] with denominator q ≤ n, written in lowest terms. For example, F5 = {0/1, 1/5, 1/4, 1/3, 2/5, 1/2, 3/5, 2/3, 3/4, 4/5, 1/1}.

A remarkable property: if a/b and c/d are adjacent Farey neighbours, their mediant (a+c)/(b+d) is the simplest fraction between them. The slider below snaps to Farey fractions of order 8, each producing a distinct Pythagorean triple from the unit circle.

The Rectangular Hyperbola: Irrational Intersections

Now replace the circle with:

xy = 1

And cut it with y = x − k, where k is a positive integer.

Substitute y = x − k into xy = 1. What quadratic appears? Is its discriminant ever a perfect square?
x(x − k) = 1  ⟹  x² − kx − 1 = 0

Here a = 1, b = −k, c = −1. The discriminant is:

Δ = k² + 4

Try it: 1+4=5, 4+4=8, 9+4=13, 16+4=20, 25+4=29... never a perfect square. So √Δ is always irrational. The metallic means are born.
