Deriving CnH2n+2

Why do alkanes have the formula CnHm with m = 2n + 2? The answer is pure graph theory. No chemistry required.

01 An alkane CnHm is a tree on n + m nodes connected, acyclic graph
02 Each C has degree 4, each H has degree 1 valency constraints
03 Sum of degrees = 4n + m count from each node
04 But sum of degrees = 2 × (number of edges) handshaking lemma
05 A tree on V vertices has V − 1 edges fundamental tree property
06 So edges = (n + m) − 1
07 Therefore: 4n + m = 2(n + m − 1) equating the two edge counts
08 4n + m = 2n + 2m − 2
09 2n + 2 = m the alkane formula
∴ CnH2n+2 QED, by graph theory alone

The three ingredients

This derivation uses exactly three facts: (1) carbon has valency 4, hydrogen has valency 1, (2) the handshaking lemma (sum of degrees = twice the number of edges), and (3) a tree on V vertices has V − 1 edges. William Clifford recognised that these graph-theoretic constraints completely determine the hydrogen count.

Cayley's Formula

Arthur Cayley, working on exactly this chemical problem, proved a theorem that became one of the jewels of combinatorics.

Labelled Trees

T(n) = nn−2

The number of labelled trees on n vertices is nn−2. This is Cayley's formula (1889), motivated by counting chemical structures.

For n = 4: T(4) = 4² = 16 labelled trees. Each vertex is distinguishable, so we count all possible tree structures with named nodes.

Proofs: Prüfer sequences (bijection with (n−2)-tuples), Kirchhoff's matrix tree theorem, double counting with Pitman's argument.

Unlabelled Trees (Alkane Isomers)

No closed form!

For alkanes, atoms of the same element are interchangeable. We need unlabelled trees with maximum degree 4. This is a much harder counting problem.

Asymptotically: ~ a · n−5/2 · bn where a ≈ 0.3183 and b ≈ 2.4833. But no exact closed-form formula is known.

Computed via Pólya enumeration theorem (1937), which accounts for symmetry by averaging over the automorphism group.

The cyanic acid example

Your slide shows the four isomers of cyanic acid: four atoms H, C, N, O arranged in a chain. With 4 labelled nodes, Cayley gives 42 = 16 labelled trees. But the atoms are distinguishable (different elements), so we only count the distinct linear permutations. The four isomers HCNO, HNCO, HOCN, HONC correspond to the four topologically distinct arrangements of these four specific atoms in a chain, taking into account the valency constraints of each element.

Why is it so hard?

Two obstacles make the unlabelled problem harder than the labelled one. First, quotienting by the symmetric group (removing labels) requires Burnside/Pólya counting, which involves summing over all permutations. Second, the degree constraint (each carbon has at most 4 bonds to other carbons) rules out most tree structures. Together, these make the count resist closed-form expression.
