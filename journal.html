Connect to Google Sheets

Entries save to a spreadsheet in your Google account. One-time setup — takes about 5 minutes.

  1. Open script.google.com → New project
  2. Open the Google Sheet you want to use → Extensions → Apps Script
  3. Paste the contents of journal_backend.gs and save
  4. Click Deploy → New deployment → Web App
  5. Execute as: Me · Access: Anyone → Deploy → Copy URL
