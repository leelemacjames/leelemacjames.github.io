Preamble I · Foundations

Bases & Powers
The grammar of scale

Before logarithms, you need to be fluent in powers. A power is repeated multiplication. Two bases dominate science: base 10 (our counting system) and base 2 (binary, doublings).

Logarithms look frightening. They are not. A logarithm answers one simple question: "What power do I raise this base to, in order to get that number?" That is all. Everything else follows from that.

Preamble II · The Logarithm

What Is a Logarithm?

The logarithm is the inverse of a power. If raising base b to power n gives x, then the log base b of x is n.

bn = x   ↔   logb(x) = n

Read it aloud: "log base b of x" = the power you raise b to, in order to reach x.

Exactly. Trace it:
log2(0.25) = log2(1/4) = log2(1/22) = log2(2−2) = −2 · log2(2) = −2 · 1 = −2.

Preamble III · The Three Rules

Three Rules That Do Everything

All log manipulation reduces to three rules. Know these and you can handle any power law.

Rule 1 · Product

log(A · B) = log A + log B

Logs turn multiplication into addition.

Rule 2 · Power

log(xn) = n · log(x)

Exponents become multipliers. Key rule for power laws.

Rule 3 · Quotient

log(A/B) = log A − log B

Division becomes subtraction.

Special Values

logb(b)=1   logb(1)=0

Log of the base = 1. Log of 1 = 0. Log of fractions < 0.

1.   log10(10 000) = ?
10 000 = 104. Count the zeros: four zeros, log = 4.
10 000 = 104 (ten thousand = ten to the four). Four zeros after the 1 means log = 4.
2.   log2(64) = ?
Count doublings: 2, 4, 8, 16, 32, 64 — six steps. 64 = 26, so log2(64) = 6.
Count the doublings from 1: 2,4,8,16,32,64 — that is six. 64 = 26, log = 6.
3.   log2(1/32) = ?
1/32 = 1/25 = 2−5. Power rule: log2(2−5) = −5. Fractions give negative logs.
32 = 25, so 1/32 = 2−5. Then log2(2−5) = −5.
4.   log10(0.0001) = ?
0.0001 = 10−4. Four decimal places means the exponent is −4.
0.0001 = 1/10 000 = 10−4. Each decimal place shifts the log by −1.
5.   log10(5 × 103) = ?   (log10(5) ≈ 0.699)
Product rule: log(5×103) = log(5)+log(103) = 0.699+3 = 3.699. The integer part is the order of magnitude; the decimal part encodes the coefficient.
log(5×103) = log(5)+3 = 0.699+3 = 3.699. In scientific notation x=a×10n, log(x)=log(a)+n.

Preamble VI · The Map
